# Lending Glossary – Personal Loans (BR)

| Nubank term (pt-BR) | Meaning | Example of use Nu | Terms users use | Comments |
|---|---|---|---|---|
| Adiantar | Pagar uma parcela do empréstimo antes da data de vencimento, o que garante desconto no valor | escolha quais parcelas você quer adiantar | | same as "antecipar" |
| Agendamento | Data de vencimento de uma parcela | pagamento agendado para o dia 20 de agosto | | same as "vencimento" |
| Análise de crédito | Avaliação feita pelo Nubank para oferta ou disponibilidade de empréstimo, ou aprovação do pedido de empréstimo, considerando diversos fatores entre o perfil e o comportamento do cliente | sujeito a análise de crédito | | |
| Antecipar | Pagar uma parcela do empréstimo antes da data de vencimento, o que garante desconto no valor | antecipando você ganha desconto | | same as "adiantar" |
| CET | Custo Efetivo Total de um serviço de crédito, como empréstimo e financiamento. São todos os encargos cobrados, como taxas, impostos e seguros. | juros e IOF anuais (CET) | | |
| Condições | Elementos necessários e importantes para a realização do empréstimo, com variáveis: taxa de juros, valor e quantidade de parcelas, data de vencimento da primeira parcela. | alterar condições do empréstimo | | |
| Contrato | Contrato e regulamento do Empréstimo Pessoal do Nubank, documento que define as condições gerais aplicáveis. | ver contrato | | |
| Custos detalhados | Especificação dos encargos financeiros relacionados ao empréstimo, como juros e impostos | ver custos detalhados | | same as "valores detalhados" |
| Data da primeira parcela | Data do vencimento da primeira cobrança referente ao empréstimo contratado | escolha a data da primeira parcela | | |
| Desconto por antecipação | Dedução do valor da parcela paga antes do vencimento | desconto por antecipação: R$x | | |
| Desistência | Cancelamento do empréstimo, que pode ser solicitado pelo cliente | Você poderá desistir do Contrato em até 7 dias corridos | | |
| Empréstimo ajustado | Quando você contrata e adiciona um novo valor ao seu empréstimo atual | taxa de juros cobrada antes do Empréstimo Ajustado | | |
| Empréstimos contratados | Contratos ativos de empréstimo | meus empréstimos contratados | | |
| IOF | Imposto Sobre Operações Financeiras | juros e IOF anuais | | |
| Juros total | Quantia cobrada sobre o valor solicitado | Juros total +R$x (xx,00%) | | |
| Número de parcelas | Quantidade de pagamentos mensais a serem debitados da conta do cliente até a quitação do empréstimo. | escolha o número de parcelas | | |
| Parcelas agendadas | Todas as parcelas a vencer | 28 parcelas agendadas | | |
| Quitação | Finalização do contrato de empréstimo pelo pagamento total da dívida. | se você quitar o empréstimo hoje, você pagará R$x | | |
| Renegociar | Reestruturação das condições do empréstimo ativo e com pagamentos em atraso. | renegociar empréstimo | | same as "reorganizar" |
| Reorganizar | Reestruturação das condições do empréstimo ativo e com pagamentos em atraso. | aqui está o resumo da reorganização do seu empréstimo de R$ 30,32 | | same as "renegociar" |
| Resumo da simulação | Descrições exatas dos termos do Empréstimo no momento da simulação: valor solicitado, quantidade de parcelas, e datas de vencimento, juros mensais, CET e total a pagar. | confira o resumo da simulação: | | |
| Saldo na conta | Valor disponível para adiantar/antecipar parcelas | saldo na conta: R$x | | |
| Simulação | Reprodução do processo de contratação de um empréstimo, mas sem efeitos práticos como a solicitação da aprovação ou contrato. | simule até R$x | | |
| Taxa aproximada | Taxa de juros provisional informada na simulação | taxa aproximada: 1,95% ao mês | | |
| Taxa de juros reduzida | Condição promocional temporária, geralmente usada em upsell | taxa de juros reduzida: 1,59% | | |
| Total a pagar | Valor total a ser pago pelo empréstimo pessoal solicitado ou aprovado, após acréscimo de todas as taxas e encargos aplicáveis | Total a pagar: R$x | | |
| Total pago | Valor já pago referente a um empréstimo pessoal ativo | total pago: R$x | | |
| Tributos | Impostos cobrados sobre o valor do empréstimo (no Nu, apenas IOF) | Tributos (IOF) | | |
| Valor a contratar | Quantia solicitada pelo usuário no momento da simulação + IOF | Valor a contratar: R$x | | |
| Valor contratado | Valor escolhido + IOF | Valor contratado: R$x | | |
| Valor desejado | Quantia solicitada pelo usuário no momento da simulação | Valor solicitado: R$x | | same as "valor escolhido" |
| Valor escolhido | Quantia solicitada pelo usuário no momento da simulação | Valor escolhido: R$x | | same as "valor desejado" |
| Valor restante | Saldo a ser pago pelo empréstimo pessoal ativo | valor restante de {{loan-name}}: R$x | | |
| Valores detalhados | Especificação dos encargos financeiros relacionados ao empréstimo, como juros e impostos | Valores detalhados | | same as "custos detalhados" |
| Vencimento | Data da cobrança referente a uma ou mais parcelas do empréstimo | Vencimento em {{data}} | | same as "agendamento" |
