# Lending Glossary – Public Payroll (BR)

| Nubank term | Meaning | Example of use Nu | Terms users use | Comments |
|---|---|---|---|---|
| Amortização | A composição das parcelas: valor emprestado (saldo devedor) + juros. | Sistema de amortização | | Existem diversos tipos de amortização. Para consignado no Nubank, usamos o sistema Price. |
| Antecipação / Antecipar | Adiantar o pagamento de uma ou mais parcelas. | Ao antecipar você ganha desconto | Antecipar, adiantar | |
| Anuência / Dar anuência | Consentimento de uma das partes envolvidas em um acordo, contrato, negociação ou transação. Utilizamos este termo jurídico por consistência, porque ele aparece na plataforma SouGov, mas podemos discutir sobre a retirada e substituição do mesmo por um termo mais cotidiano. | Você tem 3 dias para dar anuência no contrato. | | |
| Banco de origem | A instituição onde foi feito o contrato atual. Usamos este termo na hora de simular portabilidade de outra instituição financeira para Nubank. | Os valores apresentados podem mudar devido à margem disponível e da confirmação do banco de origem. | | |
| Benefício | O valor mensal que é recebido por algum órgão governamental. Pode ser uma aposentadoria, como no caso de aposentados pela INSS, ou outro tipo de quantia mensal recebida. | O dinheiro já foi depositado na conta onde você recebe seu benefício do INSS. | | Ver também "órgão" |
| Categoria de servidor | Copy onde nos referimos à sub-categoria de servidores de Estados e prefeituras | Qual é a sua categoria de servidor? | | |
| CET (Custo Efetivo Total) | O Custo Efetivo Total é o custo total de um serviço de crédito, como empréstimo ou financiamento. O CET compreende os encargos, tarifas, seguros e demais despesas que compõem a operação financeira. | Juros de 1,1% ao mês e Custo Efetivo Total (CET) de 00,00% ao ano | | |
| Consignado | Empréstimo para servidores públicos, onde a parcela é descontada mensalmente da folha de pagamento. Se escreve em minúsculas, não é nome de produto e sim, tipo. Como produto do Nubank, também escrevemos em minúscula em textos de UX e marketing (somente quando no início de frase, se escreve com a primeira letra em maiúscula). | Aqui está o resumo do seu consignado | consignado | consignado é sinônimo da palavra "empréstimo" para as pessoas elegíveis a este tipo de produto. |
| Convênio | A instituição que provém à pessoa usuária o salário de servidor público (como SIAPE) ou o benefício (como INSS). | | | na nossa jornada, utilizamos "benefício". devemos usar "convênio" em algum caso? |
| DDC (Documento Descritivo de Crédito) | Um relatório que mostra todas as informações sobre um empréstimo. Diferente do Hiscon (Extrato de empréstimo), o DDC mostra dados apenas sobre 1 contrato específico. | O que é o Documento Descritivo de Crédito (DDC) e onde encontrá-lo? | | Se escreve com letras iniciais maiúsculas, exceto a preposição "de". |
| Estados e prefeituras | Categoria de servidor público que inclui Servidores de órgãos públicos estaduais ou municipais | Estados e prefeituras | Servidores de órgãos públicos estaduais ou municipais | |
| Extrato de empréstimo | Um documento, emitido pelo INSS, que mostra o histórico de todos os empréstimos consignados ativos e anteriores de uma pessoa com este convênio. Diferente do DDC (Documento descritivo de crédito) que mostra dados sobre apenas 1 contrato específico. | Acesse o aplicativo Meu INSS para baixar o extrato atualizado em formato PDF. | Hiscon | |
| Forças armadas | Categoria de servidor público que inclui Servidores do Exército, Marinha e Aeronáutica | Forças armadas | Servidores do Exército, Marinha e Aeronáutica | |
| INSS | Um tipo de convênio. Instituto Nacional do Seguro Social, órgão público responsável pelo pagamento da aposentadoria e outros benefícios aos trabalhadores brasileiros e demais segurados – como microempreendedores individuais e contribuintes individuais. | Sou aposentado ou pensionista do INSS | INSS | |
| IOF | O Imposto sobre Operações Financeiras (IOF) é um imposto federal pago por pessoas físicas e jurídicas em qualquer operação financeira, como operações de crédito, câmbio, seguro ou operações de títulos e valores mobiliários. | Este valor, mais IOF, é somado ao seu saldo devedor atual de R$000.000,00. | | |
| Margem, margem consignável ou margem disponível | O valor máximo, dado por lei, que o cliente pode comprometer de sua renda mensal com descontos em folha de pagamento. | O valor da parcela não pode ultrapassar sua margem de R$ [*margin*] | margem | |
| Meu INSS | Aplicativo com serviços de gestão para pessoas que possuem o benefício do convênio INSS. | | Meu INSS/app do INSS | |
| Novo contrato | Todo contrato que é New Hire, que se inicia "do zero" no Nubank, que não é portabilidade nem refinanciamento de um contrato existente. | Selecione uma opção para simular: Novo contrato / Portabilidade | | |
| Oferta | Valor a emprestar com condições na base da simulação e após autorização com intermediários (Dataprev, SouGov.br, SCC e outros) | No momento não temos oferta disponível para o seu tipo de contrato e/ou benefício. | | Em Public Payroll, oferta e proposta são geralmente utilizados como sinônimos. Revisar inconsistências ou diferenças e alinhar. |
| Órgão | Instituição que emprega a pessoa usuária (diferente de convênio, é o próprio empregador do setor público, como por exemplo Exército, Estado de São Paulo, cidade de Belo Horizonte e outros) | O órgão onde você trabalha ainda não está registrado no Nubank. Te avisaremos assim que isso acontecer. | | |
| Pagamento agendado / Agendar pagamento | Parcela que foi programada pelo usuário e será paga do saldo na conta do Nubank (através da feature Autopay). | Agende o pagamento das suas parcelas e evite atrasos. | | |
| Pagamento agendado / Parcelas agendadas | [precisa ser revisto - inconsistência/duplicado] Parcela que está "na fila" das parcelas a serem descontadas da folha de pagamento normalmente. | Parcelas agendadas | | |
| Perfil de consignado/Perfil para consignado | Como perfil nos referimos ao convênio com o qual a pessoa recebe seu benefício. Pode ser Servidor Federal (SIAPE), aposentado (INSS), Forças Armadas, Servidor estadual ou municipal (convênios diversos). | Qual é o seu perfil para consignado? | | |
| Portabilidade | O processo de trocar a instituição que concedeu o crédito para outra, geralmente visando melhoria nas condições do contrato como diminuição do valor da parcela ou taxa de juros. | Pedido de portabilidade realizado com sucesso! | Portabilidade | |
| Proposta | Valor a emprestar com condições na base da simulação e após autorização com intermediários (Dataprev, SouGov.br, SCC e outros). | Confira a nossa proposta para você | | Em Public Payroll, oferta e proposta são geralmente utilizados como sinônimos. Revisar inconsistências ou diferenças e alinhar. |
| Quitação / Quitar | Pagamento total da dívida. Final do empréstimo. | Parabéns, você quitou seu empréstimo! | | |
| Refinanciamento / Refinanciar | Uma alteração de condições do empréstimo atual, com novos valores de parcela e/ou taxa de juros, e novo contrato. | Após concluir a portabilidade, você poderá refinanciar o seu contrato, adicionando até R$ 380,00 de troco. | | |
| Renegociação | Processo de renegociar uma dívida em atraso, com parcelas, multas e encargos. | [not live yet / ainda não lançado] | | |
| Saldo devedor | O valor total que resta pagar de um empréstimo, incluindo parcelas e juros. | Este valor, mais IOF, é somado ao seu saldo devedor atual de R$000.000,00. | Saldo devedor | O valor que ainda falta pagar de volta de um empréstimo. |
| SCC Consignado / app SCC Consignado | Serviço de Controle de Consignações. Um sistema centralizado de gestão para Consignatárias e Consignados. Intermediário no convênio de São Paulo. | No app SCC Consignado, encontre a opção Autorização da consulta de margem. | | |

| Servidor federal (SIAPE) | Categoria de servidor público | Servidor federal (SIAPE) | Servidores federais ativos, aposentados ou pensionistas que recebem pelo SIAPE | |

| SIAPE | Um dos convênios de Public Payroll, Sistema Integrado de Administração de Pessoal é a ferramenta que gerencia a folha de pagamento de servidores públicos. | Sou servidor federal (SIAPE) | SIAPE | |
| SIGEPE | O Sistema de Gestão de Pessoas - Sigepe - é sistema do Governo Federal, em que servidores, aposentados e pensionistas podem gerenciar assuntos relacionados com folha de pagamento, consignatárias, imposto de renda e afins. | Os sistemas SIGEPE/SouGov estão passando por uma atualização. | | |
| Simulação (Simular) | O processo que permite a visualização de valores aproximados que podem ser emprestados, através de input de dados. | Simule agora e descubra melhores condições para o seu empréstimo! | | |
| SouGov.br | Aplicativo com serviços de gestão de pessoas para servidores públicos federais ativos, aposentados, pensionistas e anistiados políticos do poder Executivo Federal civil. | Você já acessou o Sougov.br e autorizou o Nubank a acessar sua margem. | | Utilizamos somente SouGov em algumas partes do fluxo, ex Hiring error messages. Revisar consistência. |
| Troco | Referência a top-up loan. Quando a pessoa faz portabilidade ou renegocia e com a redução dos juros consegue pegar mais principal emprestado. | Após concluir a portabilidade, você poderá refinanciar o seu contrato, adicionando até R$ 380,00 de troco. | Troco | Teste de conteúdo com usuários - Troco validou uso de "troco" |

| Pagamento automático | Configuração que é possível fazer quando há falha de desconto da folha de pagamento/benefício, e o usuário deve fazer pagamento pelo app. O usuário pode configurar no app, no fluxo do loan que está identificado como "failed payment". A ativação do automatic payment serve para todos os meses restantes do empréstimo. | | | |
